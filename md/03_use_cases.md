# 使用情境與應用場景

HermesAgent 不是單純的聊天工具——它是一個可以**主動幫你做事**的 AI 代理人。以下是最常見的應用場景。

---

## 情境一：個人 AI 管家

在 Telegram 或 LINE 上，隨時用母語對話，請 AI 幫你：

- 📅 查詢並整理行事曆
- 📝 撰寫文件、Email、報告
- 🔍 搜尋資料並整理摘要
- 💡 腦力激盪、解決問題
- 📊 分析數據、製作報表

**範例對話：**
```
你：幫我寫一封婉拒合作的英文 Email，要有禮貌但清楚拒絕
AI：好的，以下是一封專業的婉拒 Email...
```

---

## 情境二：教師教學助理

教育工作者可以架設專屬 AI 助教：

- 🎓 學生問問題，AI 即時回答
- 📚 整理教材、出題、批改
- 🌐 LINE 群組 Bot，全班學生共用
- 📋 自動記錄學生問題，建立 FAQ
- 🗓️ 課程提醒、作業截止通知

> [!TIP]
> 阿亮老師本人就是用這套系統管理學生問題！透過 LINE Bot，學生可以 24 小時提問，AI 會即時回答並記錄所有問答。

---

## 情境三：辦公室自動化

接入 Google Calendar、Notion、Gmail 等工具：

- 📆 每天早上自動發送今日行程摘要
- 📬 自動整理重要 Email 並摘要
- 📋 Notion 筆記自動分類整理
- 🔔 設定排程提醒（Cron 任務）
- 📊 自動生成週報、月報

**範例 Cron 設定：**
```
每天早上 8:00 自動發送今日待辦事項到 Telegram
每週五 17:00 自動生成本週工作摘要
```

---

## 情境四：內容創作工具

創作者、自媒體人的強力助手：

- ✍️ 文章草稿、社群貼文撰寫
- 🎯 SEO 關鍵字建議
- 📹 影片腳本、字幕摘要
- 🎨 設計提示詞（Prompt）生成
- 📱 各平台內容格式轉換

---

## 情境五：程式開發助理

開發者可以接入程式工具：

- 💻 程式碼審查、重構建議
- 🐛 除錯助手，描述問題即給解法
- 📖 API 文件問答
- 🔧 Shell 指令教學
- 🚀 CI/CD 流程諮詢

---

## 情境六：LINE 群組智慧機器人

在 LINE 群組中部署 HermesAgent：

- 🌤️ 群組成員詢問天氣、交通
- 🍽️ 推薦附近餐廳
- 🎮 遊戲問答、趣味互動
- 📢 活動公告、投票整理
- 🌐 即時翻譯功能

---

## 情境七：學習與研究

學術研究者的好夥伴：

- 📚 論文摘要、重點整理
- 🔬 研究問題腦力激盪
- 📊 數據分析解讀
- 🌍 多語言資料翻譯
- 💬 概念解釋、舉例說明

---

## 不同使用者的推薦設定

| 使用者類型 | 推薦平台 | AI 大腦 | 需要 ngrok |
|-----------|---------|---------|-----------|
| 個人用戶 | Telegram | Claude / OpenAI OAuth | 否 |
| 教師 | LINE | Claude / OpenAI OAuth | 是 |
| 開發者 | Telegram | Ollama（本地）| 否 |
| 小型企業 | LINE | OpenRouter | 是 |
| 學生體驗 | Telegram | Nous Portal（免費）| 否 |

---

> [!WARNING]
> HermesAgent 目前**不適合**作為高並發商業服務的核心元件。它最適合個人、教育和小型團隊使用場景。
